# 쉽디 (Easy Design Thinking)

> 어린이·청소년도 쉽게 따라 하는 **디자인 씽킹(Design Thinking) 5단계** 워크북 웹앱

**쉽디**는 "공감 → 문제정의 → 아이디어 → 프로토타입 → 테스트"로 이어지는 디자인 씽킹 과정을
동화처럼 친근한 말투와 파스텔 UI로 안내하는 교육용 도구입니다. 별도 설치 없이 브라우저에서
바로 사용할 수 있고, 결과를 **리포트(TXT·이미지·PDF)** 로 저장할 수 있습니다.

한국어 / English 두 언어를 지원합니다.

---

## ✨ 주요 기능

| 단계 | 내용 |
|------|------|
| 1. 공감 | 도움이 필요한 대상, 상황, 감정(24종 이모지)을 기록 |
| 2. 문제정의 | "우리가 어떻게 하면...(HMW)" 문장으로 문제를 한 문장으로 정리 |
| 3. 아이디어 | 포스트잇 형태로 아이디어를 자유롭게 추가/삭제 (SCAMPER 힌트 제공) |
| 4. 프로토타입 | 펜·지우개·텍스트 그림판 + 만들기 레시피 작성 |
| 5. 테스트 | 좋았던 점 / 고칠 점 피드백 정리 |
| 리포트 | 전체 과정을 한 장의 리포트로 정리 후 TXT·이미지·PDF로 저장 |

- 🖊️ **그림판**: 펜/지우개/텍스트 도구, 색상·굵기 조절, 마우스·터치 지원
- 💾 **자동 저장**: 작성 내용은 브라우저 `localStorage`에 자동 저장되어 새로고침해도 유지
- 🌐 **다국어**: 한국어 ↔ English 즉시 전환
- 📱 **반응형**: 모바일·태블릿·PC 모두 대응

## 🆕 최근 개선 사항 (UI/UX 개선)

이번 업데이트에서 실제 사용자 피드백을 반영해 다음을 개선했습니다.

1. **상단 네비게이션 스크롤 제거**
   좁은 화면에서 단계 탭에 가로 스크롤이 생기던 문제를 해결했습니다. 이제 탭이 자동으로
   줄바꿈되어 스크롤 없이 모든 단계가 한눈에 보입니다.

2. **그림판 텍스트 삭제 기능 추가**
   캔버스에 추가한 텍스트를 삭제할 수 있습니다. 각 텍스트 위에 나타나는 빨간 **×** 버튼으로
   개별 삭제가 가능하며, 텍스트 위치는 화면 크기가 바뀌어도 유지됩니다.

3. **리포트 가독성 향상**
   섹션별 색상 카드·번호 배지, 넉넉한 여백과 큰 글씨, 아이디어 칩(chip) 스타일, 작성일 표기 등으로
   리포트를 훨씬 읽기 쉽게 다듬었습니다. 문제정의(HMW) 문장과 라벨도 언어에 맞게 표시됩니다.

4. **푸터 정리**
   "Created by. 교육뮤지컬 꿈꾸는 치수쌤" 문구 자체를 클릭하면 제작자 링크(litt.ly/chichiboo)로
   이동하도록 하나로 통합했습니다.

## 🚀 사용 방법

이 앱은 **의존성 설치나 빌드가 필요 없는 단일 HTML 파일**입니다.

- **가장 간단한 방법**: `index.html` 파일을 브라우저로 열기 (더블클릭)
- **로컬 서버로 실행** (권장, 일부 기능이 더 안정적):

  ```bash
  # Python 3
  python -m http.server 8000
  # 또는 Node
  npx serve .
  ```

  이후 브라우저에서 `http://localhost:8000` 접속

> 폰트·아이콘·CSS·내보내기 라이브러리는 CDN에서 불러오므로 **인터넷 연결**이 필요합니다.

## 🛠️ 기술 스택

- **HTML5 Canvas** — 그림판(스케치) 구현
- **Tailwind CSS** — 유틸리티 기반 스타일링
- **Vanilla JavaScript** — 프레임워크 없이 상태 관리 및 로직 구현
- **localStorage** — 작성 내용 자동 저장
